# IMDB-Movie-Reviews-Sentiment-Analysis
A production-ready sentiment analysis model for IMDB movie reviews using DistilBERT. This project implements a fine-tuned transformer-based classifier that achieves high accuracy on movie review sentiment classification.

This project fine-tunes a DistilBERT model for binary sentiment classification of IMDB movie reviews. DistilBERT is a distilled version of BERT that retains 95% of the performance while being 40% smaller and 60% faster.


Features

    Balanced Dataset Processing: Automatically creates balanced subsets to prevent model bias

    Efficient Tokenization: Optimized padding and truncation strategies

    Training Visualization: Real-time plotting of loss and accuracy metrics

    Comprehensive Evaluation: Confusion matrices, classification reports, and probability scores

    Flexible Prediction: Test on custom reviews with confidence scores

    GPU Support: Automatic device detection (CUDA/CPU)
